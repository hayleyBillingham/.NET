What is .NET

  • .NET is a free, cross-platform, open source developer platform for building many different types of applications
  • With .NET you can use mutliple languages, editors, and librries to build for web, mobile, desktop, games, and IoT
  • When languges and Libraries are used together this is what they call a developer platform
  • .NET is a collection of many parts and people that together create a powerful enviroment for building applications.
  • With .Net there are certain languages that you can you use:
    • C#
    • Visual Basic
    • F#
  • The platforms in the .net family are:
    • .NET Core: (runs anywhere!) Windows, Linux, and macOS
    • .NET Framework: websites, services, and apps on Windows
    • Xamarin/Mono: a .NET for mobile
    • .NET standard: the shared set of libraties for the above

What can I build with .NET

  • Web: build web apps and services for Windows, Linux. macOS, and Docker
  • Mobile: Use a single codebase to build native mboile apps for iOS, Android and Windows.
  • Desktop: Create beautifil and compelling desktop apps for Windows and macOS
  • Microservices: Create independently deployable microservice that run on Docker containers
  • Gaming: Develop 2D and 3D games for the most popular desktops, phones and consoles
  • Machine Learning: Add vision algorithms, speech processing, predictive models and more to your apps
  • Cloud: consume existing cloud services or create and deploy your own
    • Internet of things: Make IoT apps with native support for the Raspberry Pi, and other single board computers
