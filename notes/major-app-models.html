Major App Models

  • Web - ASP.NET Core: The framework for building server-side logic.
  • Web - ASP.NET Core: The framework for building server-side logic for web pages or web APIs
  • Web - ASP.NET Core Razor Pages: The framework for building server-generated HTML.
  • Web client - Blazor: Blazor is part of ASP.NET core. Its two modes allow for either Document Object Model (DOM) manipulation via sockets as a communication vehicle for running server-side code, or a WebAssemly implementation for running compiled C# on a browser.
  • Desktop - WinForms: A framemwork for building battleship gray Windows style applications.
  • Desktop - Windows Presentation Foundation (WPF): A framework for building dynamic desktop application that conform to different form factors. WPF allows form elements to perform movement, fades, glides and other effects with the help of rich library of animations.
  • Mobile - Xamarin: Allows .NET developers to build apps for iOS and Android devices
  • Futher more .NET providers power popular third-part and open-source gaming development environments and engines like Unity
