Create A .NET console application using VS Code

  1. Start Visual Studio Code
  2. In the Open Folder dialog create a folder and select it, then click Select Folder

    The folder name becomes the project name and the namespace by default.

  3. In the Do you trust the authers of the files in this folder?dialog, select Yes, I trust the authors.
  4. Open the Terminal in Visual Studio Code by selecting View > Terminal from the main menu.

    The Termnalopens with the command prompt in the projectFolder folder

  5. In the Terminal, enter the following command: dotnet new console --framework net6.0
  6. The project template creates a simple application that displays "Hello World" in the console window by calling the Console.WriteLine(String) method in Program.cs Console.WriteLine("Hello, World!)
